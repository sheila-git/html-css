Principais Formatações

Negrito/Destaque

Nessa frase temos um termo em negrito usando a tag STRONG.

Itálico ou Ênfase

Nesta frase temos um termo em itálico usando a tag I (não semântica).

Nesta frase temos um termo em ênfase usando a tag EM (semântica)

Texto Marcado

Podemos criar também usar um texto marcado usando a tag MARK

Texto grande e pequeno

Estamos a criar um texto grande e um texto pequeno. A tag para o texto grande BIG está obsoleta porque não tem semântica/sentido. Para aumentarmos a letra devemos fazê-lo em CSS.

Texto Rasurado

Podemos rasurar texto usando a tag DEL.

Texto Sublinhado

Podemos sublinhar texto usando a tag INS.

Texto Sobrescrito

Para inserir coisas como x 2+3 usamos a tag SUP.

Texto Subescrito

Para inserir coisas H2O usamos a tag SUB.
